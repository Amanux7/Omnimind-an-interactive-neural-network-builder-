# Omnimind – Interactive Neural Network Builder

**Built with Figma Make for the Figma Makeathon 2026**

[![Figma Make](https://img.shields.io/badge/Built%20with-Figma%20Make-00A3FF)](https://www.figma.com/make)

### Short Description
Omnimind is a professional no-code tool that lets anyone build, train, and understand neural networks through intuitive drag-and-drop interactions. It brings deep learning concepts to life with real-time simulations, smooth animations, and exportable models — making AI education visual and accessible.

### ✨ Features
- **Infinite Pannable Canvas** with natural momentum and Figma-style hand tool (hold **Spacebar**)
- **Real-time Forward Propagation** with animated signal paths and neuron firing
- **Training Simulation** with animated accuracy/loss charts
- **Keyboard Shortcuts**: `R` = Run Simulation, `T` = Train Model
- **AI Optimize Panel** with scanning orbit animation
- **Dark / Light Mode** toggle with smooth transitions
- **Onboarding Experience** with 3 instant demo networks (XOR, Digit Classifier, Regression Predictor)
- **Results Dashboard** with live metrics and predictions
- **One-Click JSON Export** of complete model structure
- **Multiplayer Cursor Simulation**
- Professional iconography and custom Cognitive Gradient color system
